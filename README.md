# Описание
**Телеграм-бот**, написанный на Python, который интегрируется с LLM, запущенной через LM Studio. Бот принимает сообщения от пользователей и отправляет их в LLM для генерации ответов, сохраняя при этом контекст диалога для каждого пользователя.

# Запуск
Для запуска бота необходимо установить библиотеки pyTelegramBotAPI, requests, jsons. Так же необходимо установить и запустить приложение **LM Studio** с загруженной языковой моделью

```bash
pip install pytelegrambotapi requests jsons
```

# Доступные команды
- `/start` — Выводит приветственное сообщение и список команд.
- `/model` — Показывает название языковой модели, используемой в LM Studio.
- `/clear` — Полностью сбрасывает историю (контекст) текущего диалога.
